# Enclosure Design Files

## CAD software used

- The OPSUM enclosure was designed entirely using [Autodesk Fusion](https://www.autodesk.com/products/fusion-360/personal).

## Folder contents

This folder hosts the OPSUM enclosure STEP file (in its latest revision).

## 3D Print settings

> Note: these are the settings I used on my `Elegoo Centauri Carbon 2` 3D printer, with the standard nozzle. They should work fine for any other printer.

The enclosure can be printed with standard settings, in **PLA** or **PETG** material:
- Layer height: 0.2mm
- Infill: 15% (minimum)
- optionally use an external brim to improve first layer adhesion
